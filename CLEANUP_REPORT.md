## 추가 권장 작업

### 즉시 실행 가능한 작업
1. **폰트 정리 실행**
   ```bash
   cd scripts
   ./cleanup-fonts.sh --remove
   ```
   - 예상 절감 용량: 약 10-15MB

2. **HTML 파일 압축**
   - HTML minification으로 추가 30-40% 크기 감소 가능

### 중장기 개선 사항
1. **모듈화**: JavaScript 코드를 별도 파일로 분리
2. **CSS 최적화**: 사용하지 않는 스타일 제거
3. **빌드 시스템**: Webpack 또는 Vite 도입 고려
4. **폰트 로딩 최적화**: 필요한 폰트만 동적 로딩
